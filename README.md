背景
==========

本仓库用于学习和记录`Linux-0.11`，通过章节的形式，将系统的一步步进行debug的过程，记录于此，且不断地更新。通过慢慢分析源码，慢慢窥探系统里面的细节，无意中被这些技术细节所打动，里面的很多细节更加可以运动到我们实际工作中，算是一种指导思想类的存在。值得每个技术人员细细品鉴。
